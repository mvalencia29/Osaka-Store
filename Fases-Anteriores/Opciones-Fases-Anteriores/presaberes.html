Tabla descriptiva etiquetas HTML

Etiqueta HTML Descripción
<html> Indica el comienzo del documento HTML.
<head> Hace referencia a la cabecera de la página, allí se suele ubicar el título.
<body> Es el cuerpo de la página, donde va lo que se ve en el navegador al cargar una web. Allí van los textos, imágenes y todo el contenido.
<a> Se usa para definir enlaces, generalmente se usa para realizar enlaces a documentos internos o externos o hacia otras páginas web.
<table> Usada para indicar la creación de una tabla.
<p> Indica la creación de un párrafo, el texto dentro de esta etiqueta forma un párrafo.
<img> Se usa para incluir imágenes dentro del contenido de la página.
<ul> Esta etiqueta crea una lista ordenada.
<strong> Se usa para resaltar el texto “negrilla”.
<u> Subraya el texto.
